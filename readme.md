## Run Locally

Clon the project

```bash
  git clone https://github.com/kioshiokamoto/notes-app
```

Go to the project route

```bash
  cd notes-app
```

Install dependencies

```bash
  yarn
```

Start Client

```bash
  yarn start
  # To open on Android: Press "a"
  # To open on iOS: Press "i"
```
